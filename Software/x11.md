### Check Wayland or X11
```sh
loginctl show-session $(awk '/tty/ {print $1}' <(loginctl)) -p Type | awk -F= '{print $2}'
```
### Change to X11
```
sed -i 's/#WaylandEnable/WaylandEnable/g' /etc/gdm3/custom.conf
```
### Turn off Screen
* Get "Output name"
```sh
a@a:~$ export DISPLAY=:0
a@a:~$ xrandr | grep " connected"
DSI-1 connected primary 1280x720+0+0 left (normal left inverted right x axis y axis) 0mm x 0mm
```
* Turn off DSI-1
```sh
a@a:~$ export DISPLAY=:0 && xrandr --output DSI-1  --off
```
